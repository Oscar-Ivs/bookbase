About BookBase

📘 What is BookBase?

BookBase is a modern, personal book tracker that lets users search for titles, log books to their collection, track reading progress, and jot down personal notes. Whether you’re a casual reader or a devoted bibliophile, BookBase helps you keep your reading life organized and meaningful.

🛠️ Why we built it

This project was created as part of a web development course to demonstrate the integration of front-end and back-end skills. From working with external APIs like Google Books, to implementing search functionality, authentication, and database handling — BookBase brings together everything in one smooth experience.

👤 Features

  • Search and add books from the Google Books API
  • Manually log your own books
  • Track your reading status (Read / Not Read)
  • Save personal notes for each book
  • Clean, responsive interface built with Bootstrap 5

🚀 What's next?

Upcoming improvements include profile customization, reading analytics, page filters, and interactive visual elements. Stay tuned for feature toggles, user preferences, and more!
